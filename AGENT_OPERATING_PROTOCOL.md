 

## Core Directive

**NEVER proceed directly to an implementation or solution without first performing the following three-stage analysis:**

1.  **Competitor & Market Research**: Use web search and internal codebase analysis to identify how industry leaders  ) solve the specific problem or implement the requested feature.
2.  **Strategic Conclusion**: Summarize the findings, identifying common patterns, UI/UX expectations, and potential pitfalls.
3.  **Unique Value-Add Proposal**: Propose specific enhancements, "wow" factors, or optimizations that go beyond standard implementations to make this platform superior.

## Execution Flow

1.  **Receive User Request.**
2.  **Conduct Web Search / Documentation Review.**
3.  **Present "Research & Proposal" Report to the User.**
4.  **Wait for User feedback or approval before writing any code.**
